> A simple yet powerful tool for monitoring websites for changes. `diff-tracker` periodically fetches a webpage, compares it to the last known version, and sends a notification if any significant changes are detected.
> 
> This is useful for tracking price updates on e-commerce sites, monitoring news websites for breaking stories, or keeping an eye on your competitors' online presence.
> 
> ## Installation
> 
> ```bash
> git clone https://github.com/ScriptStratos/diff-tracker.git
> cd diff-tracker
> pip install -r requirements.txt
> ```
> 
> ## Quick Start
> 
> To start monitoring a URL, you can run the tracker from the command line.
> 
> ```bash
> python tracker.py --url https://www.example.com --interval 60
> ```
> 
> This will check `example.com` every 60 seconds and print a message if a change is detected.
> 
> ```python
> from tracker import WebsiteTracker
> import time
> 
> def on_change_callback(url, new_content):
>     print(f"Change detected on {url}! New content length: {len(new_content)}")
> 
> tracker = WebsiteTracker(
>     urls=['https://example.com'],
>     on_change_callback=on_change_callback
> )
> 
> # Run the tracker for 5 minutes
> tracker.start()
> time.sleep(300)
> tracker.stop()
> 
> ```
> 
> ## Features
> 
> - **Content & Visual Diffing:** Tracks changes in both the HTML content and visual snapshots of the page.
> - **Configurable Interval:** Set how frequently you want to check for updates.
> - **Notification Hooks:** Easily integrate with Slack, email, or other services via callbacks.
> - **Persistent Storage:** Stores snapshots on disk to compare across runs.
> - **Lightweight:** Minimal dependencies and low resource usage.
